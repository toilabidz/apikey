Kiểm tra Key (Public)

GET /api/keys/check?key=YOUR_KEY Kiểm tra qua query string 
curl https://your-domain.vercel.app/api/keys/check?key=KM-XXXX-XXXX-XXXX-XXXX
POST /api/keys/check Kiểm tra qua body 
curl -X POST https://your-domain.vercel.app/api/keys/check \
  -H "Content-Type: application/json" \
  -d '{"key": "KM-XXXX-XXXX-XXXX-XXXX"}'

Response mẫu

Valid ✅
{ "success": true, "valid": true, "key": { "plan": "month", "expires_at": "2024-..." } }
Invalid ❌
{ "success": true, "valid": false, "reason": "KEY_EXPIRED", "message": "Key đã hết hạn" }

Admin APIs (cần Bearer Token)

POST /api/auth/login Đăng nhập admin 
{ "username": "admin", "password": "Admin@123" }
POST /api/keys/manage Tạo key mới 
{ "plan": "month", "count": 1, "label": "Premium" }
plans: day | week | month | year | lifetime | trial
